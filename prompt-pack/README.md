# Prompt Pack

Prompt packs are fallback instructions for agents or environments that cannot call the MCP server directly.

| File | Use |
| --- | --- |
| `codex-skill.md` | Codex-oriented wrapper for acting as one virtual team role. |
| `claude-code.md` | Claude Code adapter guidance and required handoff output. |
| `generic-agent.md` | Agent-neutral role behavior and traceability contract. |

Prefer MCP when available. Prompt packs are useful for manual handoff, documentation, or early adapter experiments.
