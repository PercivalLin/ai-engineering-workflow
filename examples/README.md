# Examples

Copy these examples into your agent or MCP client configuration and adjust paths.

| File | Purpose |
| --- | --- |
| `mcp-config.npm.json` | Run the MCP server through `npx` after npm publish. |
| `mcp-config.local.json` | Run the MCP server from a local source checkout. |
| `advance-workflow.request.json` | Example input for the high-level `advance_workflow` tool. |

Replace placeholder paths such as `/Users/you/.vibe-engineering` and `/absolute/path/to/target-product` before use.
